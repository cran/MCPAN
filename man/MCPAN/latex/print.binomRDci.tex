\end{Description}

## S3 method for class 'binomRDci':
print(x, digits = 4, ...)
\end{verbatim}
\end{Usage}
\begin{Arguments}
\begin{ldescription}
\item[\code{x}] an object of class "binomRDci"

\end{ldescription}
\end{Arguments}

A print out.
